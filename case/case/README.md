# Developmental / Neurodiversity Cases

このディレクトリは、発達特性（ASD / ADHD / ギフテッド等）を持つ子どもに関する  
**観察・判断・介入設計のケースログ**を集約するためのもの。

医療的診断や一般論の整理ではなく、  
**日常オペレーション上の「判断が分かれる瞬間」**を記録することを目的とする。

---

## 基本方針

- 子どもの状態を  
  **「放置してよい安全状態」**と  
  **「即時介入が必要な状態」**に分けて捉える
- 感情的評価や善悪判断を避け、  
  **観測可能なサイン**を中心に記述する
- 家庭オペレーションを崩さず、  
  **最小限の介入で鎮静・回復を促す**ことを優先する

---

## ケースの位置づけ

このディレクトリ内のケースは以下を前提とする：

- 当事者・保護者の長期観測に基づく記録
- 「正解」を出すためではなく  
  **再現性のある判断材料を残すためのログ**
- 他家庭・他事例への安易な一般化は行わない

## 収録内容の例

- 状態サイン（身体・行動・発話）
- 判断区分（放置OK / 即介入）
- 解釈メモ
- 実際に有効だった対応
- 失敗・過剰介入の記録（あれば）

---

## 注意事項

- 本ログは医療アドバイスを目的としない
- 記録の主目的は  
  **判断疲労を減らすための外部化**
- 子どもの尊厳・安全を最優先とする
