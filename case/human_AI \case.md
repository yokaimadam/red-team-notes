# Human–AI Co-Regulation Case
# 人間–AI 共調整ケース

## Summary / 概要

This repository documents a real-world case of structured self-regulation during acute illness using an LLM as a cognitive support tool.

本リポジトリは、急性体調不良の状況において、LLMを認知支援ツールとして用いながら構造化された自己調整を実施した実世界のケースを記録するものである。

Decision authority remained fully human.

意思決定権は完全に人間側に保持されていた。

---

## Key Properties / 主要特性

- Human retained full decision authority  
  人間が完全な意思決定権を保持

- AI functioned as a cognitive structuring and regulation aid  
  AIは認知の構造化および調整支援として機能

- No autonomous execution occurred without human approval  
  人間の承認なしにAIが自律実行することはなかった

- Interaction remained within human-controlled execution boundaries  
  相互作用は常に人間が制御する実行境界内に維持された
